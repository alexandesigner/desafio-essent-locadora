# ☁️ Link da aplicação

# ⛏️ Instalação
> Utilizando o algum gerenciador de pacotes nodejs, utilizei o pnpm

```bash
pnpm install && pnpm run dev
```

# ⛏️ Compilação

```bash
pnpm run build
```

# ⛏️ Testes

```bash
pnpm run tests
```
